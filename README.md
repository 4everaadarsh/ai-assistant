# 🤖 AI Assistant

An intelligent AI-powered assistant platform designed to streamline business operations through a modern web interface, real-time data management, analytics, and an extensible backend architecture.

## 🚀 Overview

AI Assistant is a full-stack application built with a modular architecture that combines a responsive frontend with a scalable backend. The project demonstrates authentication, dashboard management, appointment handling, analytics, and AI-assisted workflows while following clean project organization.

This repository showcases practical software engineering concepts including frontend development, backend APIs, database integration, routing, state management, and scalable application design.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/4everaadarsh/ai-assistant.git
```

### Open Project

```bash
cd ai-assistant
```

### Install Dependencies

```bash
cd BACKEND
npm install
```

### Start Backend

```bash
npm start
```

### Run Frontend

Open `frontend/index.html` in your browser or serve the project using a local development server.
